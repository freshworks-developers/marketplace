# Frontend App Skeleton

Freshworks Platform 3.0 frontend-only app template. Displays requester name in the ticket sidebar placeholder.

## Project Structure

```
frontend-skeleton/
├── README.md
├── app/                       # Frontend assets
│   ├── index.html             # Landing page for the app
│   ├── scripts/
│   │   └── app.js             # Frontend business logic
│   └── styles/
│       ├── images/
│       │   └── icon.svg
│       └── style.css
├── config/
│   └── iparams.json           # Installation parameters
└── manifest.json              # App metadata
```

## Running Locally

```bash
fdk run
```

## Validation

```bash
fdk validate
```
